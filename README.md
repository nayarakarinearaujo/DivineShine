# Divine Shine

## 💎 Sobre o Projeto
Divine Shine é um projeto de site para uma marca de joias exclusivas. O objetivo é oferecer uma experiência elegante e sofisticada, refletindo a essência das joias que a marca produz.

O site apresenta uma coleção de joias únicas, destacando a beleza e a personalidade de cada peça. A marca preza pela tradição artesanal e pelo design contemporâneo, garantindo qualidade e estilo atemporal.

## ✨ Funcionalidades
- Página inicial com destaque para o slogan e as coleções de joias.
- Galeria de produtos com preços e descrições.
- Seção "Sobre" com informações sobre a filosofia da marca.
- Formulário de contato para atendimento ao cliente.

## 🎨 Design
O design do site foi desenvolvido no Figma, com foco em uma estética clean e moderna. A identidade visual é composta por fontes sofisticadas e uma paleta de cores que transmite elegância e exclusividade.

## 🛠️ Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
- React
